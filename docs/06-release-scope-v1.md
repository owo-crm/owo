# Release Scope v1

Status: Draft  
Owner: Product + Engineering  
Last updated: 2026-04-02

## In scope
- Auth and business context
- Lead import and normalization
- Owner assignment
- First-contact trigger
- Task creation and follow-up tracking
- Basic lead and task dashboards
- Telegram Mini App + Web parity for core flows

## Out of scope
- Advanced billing plans and metering
- Deep customization by tenant
- Complex integrations beyond first lead source
- Enterprise admin suite

## Release criteria
- Core flows pass end-to-end on Telegram and Web.
- No blocker bugs in lead/task lifecycle.
- Response time and owner assignment metrics are observable.
